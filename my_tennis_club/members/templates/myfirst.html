¿Qué es Django?

Django es un framework web de alto rendimiento, de fácil aprendizaje y de alta productividad para la creación de aplicaciones web complejas con múltiples tópicos, desde la lógica de negocio hasta la presentación de páginas web.

El nombre de Django proviene de la palabra inglesa "llamado", que significa "llamado por el nombre". Esto se debe a que la primera versión de Django fue llamada "Django de Nombre", y se dio por supuesto que el nombre del proyecto estaba en desuso.

Funcion de Django

Django es un framework web que ayuda a los desarrolladores a crear aplicaciones web rápidamente. Esto lo hace a través de una serie de herramientas que facilitan la creación de aplicaciones web complejas, desde la lógica de negocio hasta la presentación de páginas web.
